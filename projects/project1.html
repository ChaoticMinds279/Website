Real-time Face Detection

üìå Overview

The Real-time Face Detection App is a Python-based application using OpenCV and Tkinter to detect faces through a webcam in real time. The app provides a simple graphical interface where users can start and stop face detection with a single click.

üèó Features

  • Real-time Face Detection: Uses OpenCV's Haar Cascade classifier to detect faces.
  • Graphical User Interface (GUI): Built with Tkinter for easy interaction.
  • Live Webcam Feed: Captures and processes video frames in real time.
  • Face Bounding Box: Highlights detected faces with a green rectangle.
  • Start & Stop Buttons: Allows users to control face detection easily.

  • Face detection with bounding boxes.
  • Live preview within the Tkinter window.
  • Minimalist UI for user-friendly experience.
